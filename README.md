# ForestApp - Приложение для лесного хозяйства

## Описание
ForestApp - это приложение для ведения перечётных ведомостей в лесном хозяйстве. Приложение позволяет создавать, редактировать и сохранять данные о деревьях в удобном табличном формате.

## Возможности
- ✅ Создание и управление участками
- ✅ Табличный ввод данных о деревьях
- ✅ Автозаполнение номеров деревьев
- ✅ Сохранение данных в Excel формате
- ✅ Загрузка существующих участков
- ✅ Пагинация (до 200 страниц по 50 строк)
- ✅ Навигация с помощью джойстика
- ✅ Автодополнение при вводе
- ✅ Смена тем оформления
- ✅ Горячие клавиши (Ctrl+S для сохранения)

## Установка

### Требования
- Python 3.8 или выше
- Windows 10/11

### Установка зависимостей
```bash
pip install -r requirements.txt
```

### Запуск приложения
```bash
python main.py
```

## Использование

### Главное меню
- **Перечётная ведомость** - создание нового участка для ввода данных
- **РУМ (Молодняки)** - функция в разработке
- **Сменить тему** - переключение между темами
- **Выбрать тему** - выбор из доступных тем
- **Выход** - закрытие приложения

### Работа с таблицей
1. Создайте новый участок, введя его номер
2. Заполните таблицу данными о деревьях:
   - № дерева (автозаполнение)
   - Порода
   - ж/ф (живое/фаутное)
   - шт/либо лет
   - D, см (диаметр)
   - H, м (высота)
   - Состояние
   - Модель
   - Примечания

### Навигация
- **Стрелки клавиатуры** - перемещение между ячейками
- **Джойстик** - альтернативная навигация
- **Tab** - переход к следующей ячейке
- **Enter** - переход на следующую строку

### Сохранение данных
- **Сохранить страницу** - сохранение текущей страницы
- **Сохранить отчет** - сохранение всех данных в Excel файл
- **Ctrl+S** - быстрое сохранение страницы

### Управление файлами
- **Загрузить участок** - загрузка существующего Excel файла
- **Открыть папку** - открытие папки с отчетами
- **Очистить данные** - очистка всех введенных данных

## Структура файлов
```
ForestApp/
├── main.py              # Основной файл приложения
├── requirements.txt     # Зависимости
├── README.md           # Документация
├── forest_data.db      # База данных (создается автоматически)

├── fonts/              # Шрифты
│   ├── Roboto-Medium.ttf
│   └── Roboto-Bold.ttf
├── themes/             # Темы оформления
├── reports/            # Сохраненные отчеты
└── background.jpg      # Фоновое изображение
```

## Горячие клавиши
- **Ctrl+S** - Сохранить текущую страницу
- **Стрелки** - Навигация по таблице
- **Tab** - Следующая ячейка
- **Shift+Tab** - Предыдущая ячейка
- **Enter** - Следующая строка

## Особенности
- Автоматическое создание базы данных SQLite
- Автодополнение на основе ранее введенных данных
- Поддержка до 200 страниц по 50 строк каждая
- Автоматическое сохранение конфигурации тем
- Валидация данных при вводе

## Устранение неполадок

### Приложение не запускается
1. Убедитесь, что установлен Python 3.8+
2. Установите зависимости: `pip install -r requirements.txt`
3. Проверьте наличие папки fonts с файлами шрифтов

### Ошибки при сохранении
1. Проверьте права доступа к папке reports
2. Убедитесь, что файл Excel не открыт в другой программе

### Проблемы с отображением
1. Обновите драйверы видеокарты
2. Попробуйте сменить тему оформления
